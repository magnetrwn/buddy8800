Detailed Description

template<bool construct_then_write_lock>
class data_card< construct_then_write_lock >

A card that holds a fixed amount of data.

Parameters
start_adrThe starting address of the card.
capacityThe size in bytes of the card starting from the start address.
construct_then_write_lockWhether the card should be write-locked after construction.

This class can be used to create cards that hold a fixed amount of data. It allows copying data to the card immediately after construction, and can be write-locked after construction if needed. Write locking can also be toggled at any time.

When setting up the card by copying data to it, the capacity of it will be determined by the size of the provided data. This might be a bit unrealistic in address range.

Note
For convenience, use the aliases ram_card and rom_card instead of this class.
Warning
Out of range addresses are not checked, they should be checked by the bus instead, to avoid calling in_range() twice.
